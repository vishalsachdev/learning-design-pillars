## Pillar 1: Clear, Purposeful Structure

| ID | Criterion | Canvas Check | Weight |
|----|-----------|--------------|--------|
| 1.1.1 | Small segments | Items per module <= 10 | High |
| 1.1.2 | Self-contained | Module dependencies minimal | Medium |
| 1.1.3 | Logical sequencing | Prerequisites set appropriately | High |
| 1.1.4 | Hierarchical layouts | Pages use H1/H2/H3 | Medium |
| 1.1.5 | Structural cues | Headers, labels present | Medium |
| 1.2.1 | Standardized features | Consistent naming conventions | High |
| 1.2.2 | Integrated format | Resources embedded vs linked | Medium |
| 1.2.3 | Program continuity | References to prior modules | Low |
| 1.3.1 | Learning objectives | Visible in modules/assignments | High |
| 1.3.2 | Openers and closers | Overview/summary pages exist | Medium |
| 1.3.3 | Lesson alignment | Rubrics match objectives | High |
| 1.3.4 | Expectation setting | Rubrics attached, samples provided | High |
| 1.3.5 | Course introduction | Syllabus complete, welcome content | High |
| 1.4.1 | Spaced exposure | Concepts revisited in later modules | Low |
| 1.4.2 | Learner control | Optional/supplementary content | Medium |
